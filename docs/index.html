kernel.css

Simplicity done right

Download

Built for any kind of job

Whether you're making a website, web app or a dashboard app, this CSS library got you covered!

Far from reaching its potential

If you like this project and would like to help out, you may do so on GitHub. This project is ever improving.

Simple to use

The kernel.css library is designed for everyone from the beginner to the intermediate. It makes everybody's job easier.

Lightweight

We aim to make this a super lightweight, fully fledged library that wont slow down your website's loading times.

A solid foundation for your project

The kernel.css library does not aim to do all the work for you by any means, it is meant to be a generic foundation that can be used for any type of project.

Compatible with other CSS frameworks

Because kernel.css supports prefixes for all it's css classes (ke-), it can be used with most other CSS frameworks. Mix and match

Installation

Add the following css tag to your html page.

<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/christian-dale/kernel.css@2.0.0-beta.1/build/kernel.min.css" />

Prefix

Kernel.css uses prefixing of classes with the ke- prefix.

Base

The ke-base class contains the styles for typography, links, headings and paragraphs. Note: You are free to wrap this class for any section you wish.

<body class="ke-base">
...
</body>
<body>
    <div>
        <h1>Heading</h1>
    </div>
    <div class="ke-base">
        ...
    </div>
</body>

Alternatively you can use the ke-modern class for a more stylized look.

Headings

Buttons

<!-- Default Button -->
<button class="ke-button ke-btn-default">Button</button>

<!-- Primary Button -->
<button class="ke-button ke-btn-primary">Button</button>

<!-- Safe Button -->
<button class="ke-button ke-btn-safe">Button</button>

<!-- Warning Button -->
<button class="ke-button ke-btn-warning">Button</button>

<!-- Risk Button -->
<button class="ke-button ke-btn-risk">Button</button>

<!-- Minimal Button -->
<button class="ke-button ke-btn-minimal">Button</button>

<!-- Large button -->
Just append the "ke-btn-lg" class to any of the button types.

You may also use the "ke-button" class with anchor tags and button inputs,
and you can extend the class if you want to customize your own buttons.

Notices

Use notices to inform your users of something important. There are multiple types of notices: Default, affirmation, info, warning and alert.

Notice: This is a default notice.
Notice: This is an affirmation.
Notice: This is an info notice.
Warning: This is a warning.
Alert: This is an alert! 
<div class="ke-notice">This is a default notice.</div>
<div class="ke-notice ke-safe">This is an affirmation.</div>
<div class="ke-notice ke-info">This is an info notice.</div>
<div class="ke-notice ke-warning">This is a warning.</div>
<div class="ke-notice ke-alert">This is an alert!</div>

Adding a close button to the notice

kernel.css is a CSS-only library, but for your convenience, here's how you can add a simple close button using JavaScript.

× Notice: This is a default notice. 
<div class="ke-notice">
    <span class="ke-notice-close" onclick="this.parentElement.remove();">×</span>
    <strong>Notice: </strong> This is a default notice.
</div>

Progressbar

The progress of the progressbar is controlled by setting the "width" style attribute.

<div class="ke-progressbar">
    <div class="ke-progress" style="width: 25%"></div>
</div>

Code

The code class is used for displaying code. The first way to do this is inline code (like this), and code blocks: 

Inline code
<code class="ke-code">...</code>
Code block
<pre class="ke-code ke-code-block"><code>...</code></pre>

Note: Code highlighting is not part of kernel.css.

Utils

Kernel.css comes bundled with a few useful utility classes.

<!-- Anchor -->
<a class="ke-anchor" href="#">

<!-- Horizontal Rule (hr) -->
<hr class="ke-hr">

<!-- Section -->
Gives a set margin to an element. Used to seperate content.
<div class="ke-section"></div>

<!-- No-shadow -->
Removes all shadows from an element.
<div class="ke-noshadow"></div>

<!-- Vertical-center -->
Vertically center an element.
<div class="ke-vertical-center">...</div>

<!-- Center (horizontal) -->
Center an element horizontally.
<div class="ke-center"></div>

<!-- Circle -->
Make an element circular.
<div class="ke-circle"></div>

<!-- Horizontal Rule (hr) -->
<hr class="ke-hr">

<!-- Display only on small screen sizes -->
<div class="ke-visible-sm">...</div>

<!-- Display only on large screen sizes -->
<div class="ke-visible-lg">...</div>

<!-- Autowrap a piece of text -->
<p class="ke-autowrap">...</p>
